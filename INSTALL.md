# 설치 가이드

## 1. 내려받기

[Releases](../../releases/latest) 에서 `oh-my-grid-x.x.x.dmg` 를 받습니다.

## 2. Applications 로 드래그

`.dmg` 를 더블클릭해 열면 창이 뜹니다. **`oh-my-grid` 아이콘을 옆의 `Applications` 폴더로 드래그**하세요.

## 3. 처음 한 번 보안 우회

자체서명 앱이라 처음 실행 시 막힐 수 있습니다.

1. 앱을 한 번 더블클릭합니다(경고가 떠도 괜찮습니다).
2. **시스템 설정 → 개인정보 보호 및 보안** 으로 갑니다.
3. 아래쪽 보안 항목에 `"oh-my-grid"을(를) 열 수 없습니다…` 메시지와 함께 **`그래도 열기`** 버튼이 보입니다. 누릅니다.
4. 다시 **열기** 를 확인합니다.

## 4. 손쉬운 사용(접근성) 권한 켜기

oh-my-grid 가 창을 옮기려면 접근성 권한이 필요합니다.

1. 첫 실행 시 권한 요청 창이 뜹니다. **시스템 설정 열기** 를 누릅니다.
2. **개인정보 보호 및 보안 → 손쉬운 사용** 목록에서 **`oh-my-grid` 를 ON** 으로 켭니다.
3. **앱을 완전히 종료한 뒤 다시 실행**합니다(권한은 재시작 후 적용됩니다).

## 5. 써 보기

1. 아무 창이나 타이틀바를 잡고 드래그를 시작합니다.
2. 드래그 도중 오른쪽 버튼을 한 번 클릭하면 그리드가 켜집니다.
3. 왼쪽 버튼을 누른 채 셀을 가로질러 움직인 뒤 왼쪽 버튼을 놓으면 창이 스냅됩니다.

---

### 자동 업데이트

한 번 설치하면 이후 버전은 **Sparkle** 이 자동으로 확인해 알려줍니다. 보안 우회를 다시 할 필요는 없습니다.
