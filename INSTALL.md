# 설치 가이드

## 1. 내려받기

Homebrew를 쓰고 있다면 아래 명령으로 설치할 수 있습니다.

```sh
brew install --cask canine89/tap/oh-my-grid
```

직접 설치하려면 [Releases](../../releases/latest) 에서 `oh-my-grid-x.x.x.dmg` 를 받습니다.

## 2. Applications 로 드래그

`.dmg` 를 더블클릭해 열면 창이 뜹니다. **`oh-my-grid` 아이콘을 옆의 `Applications` 폴더로 드래그**하세요.

## 3. 앱 실행

`Applications` 폴더에서 `oh-my-grid` 를 실행합니다.

배포용 DMG는 **Apple Developer ID 서명 및 공증**을 거쳤습니다. 이전 안내와 달리 보통 `그래도 열기` 보안 우회가 필요하지 않습니다. macOS가 확인 대화상자를 띄우면 안내에 따라 열면 됩니다.

## 4. 손쉬운 사용(접근성) 권한 켜기

oh-my-grid 가 창을 옮기려면 접근성 권한이 필요합니다.

1. 첫 실행 시 권한 요청 창이 뜹니다. **시스템 설정 열기** 를 누릅니다.
2. **개인정보 보호 및 보안 → 손쉬운 사용** 목록에서 **`oh-my-grid` 를 ON** 으로 켭니다.
3. 권한을 켠 뒤 잠시 기다리면 바로 동작합니다.

## 5. 써 보기

1. 아무 창이나 타이틀바를 잡고 드래그를 시작합니다.
2. 드래그 도중 오른쪽 버튼을 한 번 클릭하면 그리드가 켜집니다.
3. 왼쪽 버튼을 누른 채 셀을 가로질러 움직인 뒤 왼쪽 버튼을 놓으면 창이 스냅됩니다.
