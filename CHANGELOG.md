# 변경 이력

## 1.1.2

- 드래그 감지 때문에 다른 앱 클릭/입력이 지연되던 문제 수정 (창 조회를 첫 드래그 시점에 비동기로 처리, 순수 클릭에는 영향 없음)

## 1.1.1

- 그리드 스냅이 Dock·상단 메뉴바 영역을 침범하던 문제 수정 (사용 영역 기준으로 셀 계산)

## 1.1.0

- 가장자리 절반 스냅 추가: 창을 그냥 끌어서 화면 **좌·우·아래** 끝에 대면 절반, **위** 끝에 대면 최대화
- 텍스트 선택·리사이즈와 구분(창이 실제로 이동할 때만 발동) · 메뉴바/Dock 영역은 침범하지 않음
- 설정에서 "가장자리 절반 스냅" 켜기/끄기 가능 (기본 켜짐)

## 1.0.2

- 새 앱 아이콘 적용 (그리드 + 캐릭터 로고)

## 1.0.1

- 동작 방식 변경: 오른쪽 버튼을 계속 누르지 않고, 드래그 도중 **오른쪽 클릭 한 번**으로 그리드를 켭니다
- 왼쪽 버튼을 놓으면 스냅 완료 · 다시 오른쪽 클릭하거나 Esc로 취소

## 1.0.0

- 첫 릴리스 🎉
- 창을 드래그하는 도중 우클릭하면 화면에 그리드 가이드 표시
- 셀 블록을 끌어 놓으면 창이 해당 영역으로 스냅(이동·리사이즈)
- 그리드 열·행 수 설정(기본 6×4)
- 메뉴바 상주 · Sparkle 자동 업데이트
